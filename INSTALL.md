# Installation Guide

## Caveman Mode

Caveman mode is a token-efficient communication style that reduces token usage by ~75% while maintaining technical accuracy.

### How to Use

- **/caveman** - Enter full caveman mode (default)
- **/caveman-lite** - Lite intensity
- **/caveman-full** - Full intensity  
- **/caveman-ultra** - Ultra intensity
- **/caveman-help** - Get reference card for all modes

### Benefits

- Ultra-compressed communication
- ~75% token reduction
- Maintains full technical accuracy
- Available intensities: lite, full, ultra, wenyan-lite, wenyan-full, wenyan-ultra

### Auto-Trigger Conditions

Caveman mode activates automatically when:
- User says "caveman mode", "talk like caveman", "use caveman"
- Token efficiency is requested
- Using `/caveman` command

### Commands

- `/caveman-help` - Get quick reference card
- `/caveman-stats` - View usage statistics
- `/caveman-review` - Review in caveman mode
- `/caveman-commit` - Generate compressed commit messages
